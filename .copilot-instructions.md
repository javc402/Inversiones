# Copilot Instructions - Proyecto Inversiones

## 🎯 Contexto del Proyecto

Este es un proyecto de migración de un sistema Excel a una aplicación web moderna. El proyecto se llama **Inversiones** y es un sistema de gestión de operaciones de trading.

## 📊 Información Técnica

- **Stack Frontend:** React 18, TypeScript, Recharts, Tailwind CSS
- **Backend:** Supabase (PostgreSQL + API REST)
- **Autenticación:** Google OAuth (única opción)
- **Versionado:** Git Flow (dev → release → main)
- **Hosting:** GitHub Pages (frontend) + Supabase (backend)
- **Lenguaje:** Español (interfaz) + Inglés (código)

## 📁 Estructura Principal

```
/prototipos     - Prototipo HTML/CSS estáticos (LOGIN + DASHBOARD)
/analisis       - Documentación técnica del proyecto
/Datos Originales - Excel original con datos reales
/.instructions.md - Guía para desarrolladores
/README.md      - Documentación principal
```

## 🚀 PRÓXIMO PASO: CREAR REACT APP

El siguiente paso es crear la aplicación React desde los prototipos:

```bash
npm create vite@latest frontend -- --template react-ts
cd frontend
npm install @supabase/supabase-js react-router-dom recharts axios
npm run dev
```

## 📝 Convenciones de Código

- **Componentes React:** PascalCase
- **Variables/funciones:** camelCase
- **Constantes:** UPPER_CASE
- **Tipos TypeScript:** No usar `any`
- **Commits:** `feat:`, `fix:`, `docs:`, etc.

## 🌿 Git Workflow

1. Crear rama: `git checkout -b feature/nombre`
2. Commit: `git commit -m "feat: descripción"`
3. Push: `git push origin feature/nombre`
4. PR hacia `dev` (no main)
5. Después de merge, delete branch local
