💡 Cómo cambiar la paleta
  • Todos los colores están centralizados en assets/css/paleta.css
  • Usa variables CSS (ej: var(--color-primary)) en todo el código
  • Para cambiar el azul: edita --color-primary-dark y otros colores en la sección :root
  • Los cambios se aplican automáticamente en todo el sitio

    ✨ Características de los Prototipos
  • Paleta de colores centralizada: Todos los colores se definen en assets/css/paleta.css
  • Variables CSS: Fácil de cambiar colores, espaciados y tipografía
  • Responsive: Se adapta a mobile, tablet y desktop
  • Accesible: Contraste de colores y navegación clara
  • Componentes reutilizables: Botones, formularios, cards, tablas
  • Sin dependencias: Solo HTML y CSS puro
